# Generative AI & Prompt Engineering

## 🎯 Module objective

Master generative AI techniques (GANs, VAEs, Diffusion) and prompt engineering to create synthetic content and optimize interactions with LLMs.

## 💡 What are you going to achieve

- Implement GANs (vanilla, DCGAN, conditional, WGAN) to generate images
- Build Variational Autoencoders with reparameterization trick
- Understand diffusion models and denoising process
- Apply prompt engineering advanced (zero-shot, few-shot, chain-of-thought)
- Identify and mitigate mode collapse in GANs
- Explore latent spaces and generate interpolations

## 📂 Structure interna

- **theory/**: fundamentals (GANs, VAEs, diffusion, prompt engineering strategies)
- **examples/**: guided examples with PyTorch
- **practices/**: Exercises hands-on
  - 01: GANs (vanilla GAN, DCGAN, conditional GAN, mode collapse, WGAN, Pix2Pix)
  - 02: VAE and Diffusion (VAE implementation, latent interpolation, β-VAE, diffusion basics, denoising)
- **mini-project/**: complete generator (images or text)
- **evaluation/**: checklist and rubric
- **notes/**: experiments with generative architectures

## 📋 Prerrequisitos

- Module 04: Deep Learning Basics (PyTorch, training loops)
- Module 05: NLP & LLMs (for advanced prompt engineering)
- Familiarity with backpropagation and optimization
